Журнал изменений

История обновлений и улучшений языка Taurine

v1.0.5

17 марта 2026

🎉 Нововведения

  • Улучшена производительность интерпретатора на 15%
  • Добавлена поддержка новых криптографических функций
  • Улучшена обработка ошибок в JSON парсере

🐛 Исправления

  • Исправлена ошибка с утечкой памяти при работе с большими массивами
  • Исправлен баг с некорректной обработкой nil-safe операторов
  • Устранена проблема с кодировкой UTF-8 в строковых функциях

📚 Документация

  • Обновлены примеры кода в документации
  • Добавлены новые руководства по встраиванию
  • Улучшена навигация по сайту документации

v1.0.0

16 января 2026 Стабильный релиз

🎉 Стабильный релиз!

Первый стабильный релиз языка программирования Taurine.

✨ Добавлено

  • let/const объявления переменных
  • f-строки для интерполяции строк
  • Multi-return функции с деструктуризацией
  • break/continue в циклах
  • Nil-safe операторы (?.)
  • JSON-стиль таблиц ({key: val})
  • Диапазоны (1..10)
  • Присваивание по индексу массива (arr[i] = value)
  • Параметры по умолчанию в функциях
  • Функциональные литералы (анонимные функции)
  • Rust API — Встраивание в Rust приложения
  • C API — Использование из C/C++ программ

📚 Стандартная библиотека

  • std/json.tau — Парсинг/сериализация JSON
  • std/http.tau — HTTP клиент
  • std/crypto.tau — Криптографические функции
  • std/date.tau — Дата/время
  • std/regex.tau — Регулярные выражения
  • std/array.tau — Утилиты для массивов
  • std/string.tau — Строковые утилиты
  • std/io.tau — Файловый ввод/вывод
  • std/math.tau — Математические функции
  • std/os.tau — Функции ОС

⚡ Производительность

  • Замена Arc<Mutex<>> на Rc<RefCell<>> для ускорения в 2-3 раза
  • Constant folding оптимизатор
  • Dead code elimination

🔄 Изменено

  • Удалён менеджер пакетов (фокус на ядре языка)
  • Упрощён CLI интерфейс
